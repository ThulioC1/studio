# **App Name**: Consulta CNPJ Pro

## Core Features:

- Busca por CNPJ: Permite ao usuário inserir um número de CNPJ para pesquisar informações da empresa.
- Exibição de Dados da Empresa: Apresenta de forma clara e estruturada os dados obtidos da empresa (ex: razão social, endereço, status, atividades principais).
- Serviço de Busca de Dados CNPJ: Integração com uma API externa (pública ou comercial) para consultar e obter os dados das empresas a partir do CNPJ informado.
- Feedback de Status: Exibe indicadores de carregamento, mensagens de erro para CNPJ inválido ou empresa não encontrada, e feedback de sucesso.
- Insights Essenciais da Empresa (AI): Uma ferramenta de IA generativa para sumarizar informações-chave da empresa pesquisada, destacando detalhes cruciais como a situação cadastral ou um resumo das atividades principais com base nos dados públicos disponíveis.
- Cópia Rápida de Informações: Funcionalidade para copiar rapidamente campos específicos dos dados da empresa para a área de transferência.

## Style Guidelines:

- Esquema de cores claro. A cor primária, um azul-acinzentado ( #376482 ), foi escolhida para evocar profissionalismo, confiança e clareza, alinhando-se com a natureza séria dos dados de negócios.
- A cor de fundo é um cinza muito claro com um toque do matiz principal ( #F5F7F8 ), fornecendo um ambiente limpo e neutro que facilita a leitura de informações detalhadas.
- Um tom vibrante de ciano ( #21D4DA ) é usado como cor de destaque para elementos interativos e chamadas à ação, criando um contraste eficaz e modernidade.
- A fonte 'Inter' (sans-serif) é recomendada para todos os textos. Sua natureza moderna e legível garante que os dados da empresa sejam apresentados de forma clara e concisa, adequada tanto para títulos quanto para conteúdo.
- Utilizar ícones vetoriais simples e lineares, como lupas para pesquisa, ou ícones de documento/pasta para dados empresariais, mantendo uma estética limpa e funcional.
- Um layout responsivo e baseado em grade para exibir os resultados da pesquisa, garantindo uma experiência de usuário consistente em diferentes dispositivos. O campo de busca será proeminente, seguido por blocos de informações bem definidos para os dados da empresa.
- Animações sutis e funcionais, como um indicador de carregamento discreto ao buscar o CNPJ ou transições suaves na exibição de detalhes, para melhorar a percepção de agilidade sem distrair o usuário.
